// 核心特性

为生产环境而生

从实时通信到安全防护，LingFlow 提供构建企业级 AI 聊天应用所需的一切能力

⚡

流式透明响应

客户端实时接收 AI 思考过程与最终响应，system_thinking + system_response 双阶段流式推送，完全可控可追溯。

📦

S3 技能热加载

技能以 Markdown 文件存储于 S3，无需重新编译即可热更新。支持动态加载、关键词检索、智能匹配。

🔗

事件溯源架构

所有状态变更以不可变事件记录，支持完整审计追踪、会话回放、时间旅行调试。

🛡️

七层安全防护

Token 认证、Origin 白名单、IP 限流、速率限制、提示注入检测、内容审查、Guardrail 集成。

🤖

AI 技能创建

通过 #create_skill 命令让 AI 自动生成技能，两阶段提交 + 事务锁 + 速率限制确保安全可靠。

☁️

多云部署支持

支持本地服务器、Docker、Kubernetes、AWS Lambda + API Gateway 多种部署方式，灵活扩展。
